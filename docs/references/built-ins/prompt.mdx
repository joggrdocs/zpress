# Prompt

The `<Prompt>` component renders a copyable AI prompt block with a sparkle icon header and one-click copy button. Designed for sharing prompts, instructions, and reusable text snippets.

## Basic

**Code**

```mdx
import { Prompt } from '@theme'

<Prompt>Explain how zpress generates sidebar navigation from the config file.</Prompt>
```

**Output**

<Prompt>Explain how zpress generates sidebar navigation from the config file.</Prompt>

## Multi-line

**Code**

```mdx
<Prompt>
  You are a documentation expert. Review the following API endpoint and generate a clear, concise
  description that includes: - What the endpoint does - Required parameters - Example response
</Prompt>
```

**Output**

<Prompt>
  You are a documentation expert. Review the following API endpoint and generate a clear, concise
  description that includes: - What the endpoint does - Required parameters - Example response
</Prompt>

## Props

| Prop       | Type        | Required | Default | Description            |
| ---------- | ----------- | -------- | ------- | ---------------------- |
| `children` | `ReactNode` | yes      | —       | Prompt text to display |
