# Prompt

The `<Prompt>` component displays an AI prompt with a description, action buttons, and an expandable code view. The first action renders as a full button; additional actions appear in a dropdown menu.

## Basic

**Code**

```mdx
import { Prompt } from '@theme'

<Prompt description="Generate clear, concise documentation.">
  You are a technical writing assistant. Write documentation that is clear, accurate, and concise.
  Use second-person voice and active verbs.
</Prompt>
```

**Output**

<Prompt description="Generate clear, concise documentation.">
  You are a technical writing assistant. Write documentation that is clear, accurate, and concise.
  Use second-person voice and active verbs.
</Prompt>

## With multiple actions

The first action is the primary button. Additional actions go into a dropdown.

**Code**

```mdx
<Prompt
  description="Review code for security vulnerabilities."
  icon="pixelarticons:shield"
  actions={['copy', 'cursor', 'chatgpt', 'claude']}
>
  You are a security auditor. Review the provided code for OWASP Top 10 vulnerabilities. Flag any
  issues with severity level and remediation steps.
</Prompt>
```

**Output**

<Prompt
  description="Review code for security vulnerabilities."
  icon="pixelarticons:shield"
  actions={['copy', 'cursor', 'chatgpt', 'claude']}
>
  You are a security auditor. Review the provided code for OWASP Top 10 vulnerabilities. Flag any
  issues with severity level and remediation steps.
</Prompt>

## All actions

**Code**

```mdx
<Prompt
  description="Generate API endpoint documentation."
  icon="pixelarticons:article"
  actions={['copy', 'cursor', 'vscode', 'chatgpt', 'claude']}
>
  You are a documentation expert. Review the following API endpoint and generate a clear description
  with parameters and examples.
</Prompt>
```

**Output**

<Prompt
  description="Generate API endpoint documentation."
  icon="pixelarticons:article"
  actions={['copy', 'cursor', 'vscode', 'chatgpt', 'claude']}
>
  You are a documentation expert. Review the following API endpoint and generate a clear description
  with parameters and examples.
</Prompt>

## Props

| Prop          | Type             | Required | Default                    | Description                                       |
| ------------- | ---------------- | -------- | -------------------------- | ------------------------------------------------- |
| `description` | `string`         | no       | —                          | Short description of what the prompt does         |
| `icon`        | `string`         | no       | `'pixelarticons:sparkles'` | Iconify icon ID                                   |
| `actions`     | `PromptAction[]` | no       | `['copy']`                 | Action buttons (first = primary, rest = dropdown) |
| `children`    | `ReactNode`      | yes      | —                          | The raw prompt text (shown in expanded view)      |

### PromptAction

| Value       | Description                            |
| ----------- | -------------------------------------- |
| `'copy'`    | Copy prompt text to clipboard          |
| `'cursor'`  | Copy for Cursor IDE                    |
| `'vscode'`  | Copy for VS Code                       |
| `'chatgpt'` | Open in ChatGPT with prompt pre-filled |
| `'claude'`  | Open in Claude with prompt pre-filled  |
