# IDE Window

The `<IDEWindow>` component wraps code in an editor-style window with file tabs in the title bar. Built on top of [DesktopWindow](/references/built-ins/desktop-window).

## Single file

Define code as an `export const` and pass it via the `code` prop. This preserves indentation (inline template literals in JSX attributes are dedented by the MDX compiler).

**Code**

````mdx
import { IDEWindow } from '@theme'

export const configExample = `import { defineConfig } from '@zpress/kit'

export default defineConfig({
  title: 'My Docs',
  description: 'A documentation site',
  sections: [
    {
      title: 'Getting Started',
      path: '/getting-started',
      include: 'docs/getting-started/*.md',
    },
  ],
})`

<IDEWindow
  files={[{ name: 'zpress.config.ts', active: true }]}
  lang="ts"
  code={configExample}
/>
````

**Output**

    },
  ],

<IDEWindow

/>

## Multiple tabs

Show multiple file tabs to indicate context. The `active` tab is visually highlighted:

export const multiTabExample = `export const config = {
  port: 3000,
  host: 'localhost',
}`

**Code**

````mdx
export const multiTabExample = `export const config = {
  port: 3000,
  host: 'localhost',
}`

<IDEWindow
  files={[
    { name: 'index.ts' },
    { name: 'config.ts', active: true },
    { name: 'types.ts' },
  ]}
  lang="ts"
  code={multiTabExample}
/>
````

**Output**

<IDEWindow
  files={[
    { name: 'index.ts' },
    { name: 'config.ts', active: true },
    { name: 'types.ts' },
  ]}
  lang="ts"
  code={multiTabExample}
/>

## With path-style filenames

Use full paths for clarity in monorepo contexts:

export const pathExample = `import { resolveConfig } from './config'
import { generateSidebar } from './sidebar'
import { writeManifest } from './manifest'

export const sync = async (root: string) => {
  const config = await resolveConfig(root)
  const sidebar = generateSidebar(config.sections)
  await writeManifest(root, sidebar)
}`

**Code**

## Props

### IDEWindow

| Prop | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `files` | `IDEFileTab[]` | yes | — | File tabs in the title bar |
| `code` | `string` | no | — | Code string to display (define as `export const` to preserve indentation) |
| `lang` | `string` | no | — | Language identifier (e.g. `'ts'`, `'json'`) |
| `children` | `ReactNode` | no | — | Fallback content when `code` is not provided |

### IDEFileTab

| Prop | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `name` | `string` | yes | — | Filename displayed in the tab |
| `active` | `boolean` | no | `false` | Whether this tab is selected |
