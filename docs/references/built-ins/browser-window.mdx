# Browser Window

The `<BrowserWindow>` component wraps content in a macOS-style browser frame with traffic-light dots, optional tabs, and an optional URL bar below the header. Built on top of [DesktopWindow](/references/built-ins/desktop-window).

## With URL

**Code**

```mdx
import { BrowserWindow } from '@theme'

<BrowserWindow url="https://docs.example.com">

# Welcome to My Docs

Get started with our platform in minutes.

</BrowserWindow>
```

**Output**

<BrowserWindow url="https://docs.example.com">

# Welcome to My Docs

Get started with our platform in minutes.

</BrowserWindow>

## With tabs and URL

**Code**

```mdx
<BrowserWindow
  tabs={[{ name: 'docs.example.com', active: true }, { name: 'github.com' }]}
  url="https://docs.example.com/guides"
>

Content with browser tabs and a URL bar.

</BrowserWindow>
```

**Output**

<BrowserWindow
  tabs={[{ name: 'docs.example.com', active: true }, { name: 'github.com' }]}
  url="https://docs.example.com/guides"
>

Content with browser tabs and a URL bar.

</BrowserWindow>

## Without URL

Omit the `url` prop for a minimal frame:

**Code**

```mdx
<BrowserWindow>

Content without a URL bar — useful for generic previews.

</BrowserWindow>
```

**Output**

<BrowserWindow>

Content without a URL bar — useful for generic previews.

</BrowserWindow>

## Wrapping components

Combine with other zpress components to preview how they render:

**Code**

```mdx
import { BrowserWindow, SectionCard, SectionGrid } from '@theme'

<BrowserWindow url="https://docs.example.com/guides">

<SectionGrid>
  <SectionCard
    href="/guides/setup"
    title="Setup"
    description="Install and configure your project."
    icon="pixelarticons:cloud-upload"
    iconColor="blue"
  />
  <SectionCard
    href="/guides/deploy"
    title="Deploy"
    description="Ship to production in one command."
    icon="pixelarticons:speed-fast"
    iconColor="green"
  />
</SectionGrid>

</BrowserWindow>
```

**Output**

<BrowserWindow url="https://docs.example.com/guides">

<SectionGrid>

</SectionGrid>

</BrowserWindow>

## Props

| Prop       | Type          | Required | Default | Description                                       |
| ---------- | ------------- | -------- | ------- | ------------------------------------------------- |
| `url`      | `string`      | no       | —       | URL displayed in the address bar below the header |
| `tabs`     | `WindowTab[]` | no       | —       | Tabs rendered in the title bar                    |
| `children` | `ReactNode`   | yes      | —       | Content rendered inside the window body           |
