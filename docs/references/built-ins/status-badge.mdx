# Badge

The `<Badge>` component renders inline labels for marking features, API endpoints, or documentation sections with status or category information.

## Variants

**Code**

```mdx
import { Badge } from '@theme'

- <Badge variant="new">New</Badge> — just shipped
- <Badge variant="beta">Beta</Badge> — testing phase
- <Badge variant="experimental">Experimental</Badge> — may change
- <Badge variant="stable">Stable</Badge> — production ready
- <Badge variant="deprecated">Deprecated</Badge> — avoid using
- <Badge variant="info">Info</Badge> — informational
- <Badge variant="warning">Warning</Badge> — heads up
```

**Output**

- <Badge variant="new">New</Badge> — just shipped
- <Badge variant="beta">Beta</Badge> — testing phase
- <Badge variant="experimental">Experimental</Badge> — may change
- <Badge variant="stable">Stable</Badge> — production ready
- <Badge variant="deprecated">Deprecated</Badge> — avoid using
- <Badge variant="info">Info</Badge> — informational
- <Badge variant="warning">Warning</Badge> — heads up

## Inline with headings

```mdx
## Authentication <Badge variant="beta">Beta</Badge>
```

## Props

| Prop       | Type           | Required | Default  | Description                      |
| ---------- | -------------- | -------- | -------- | -------------------------------- |
| `variant`  | `BadgeVariant` | no       | `'info'` | Visual variant controlling color |
| `children` | `ReactNode`    | yes      | —        | Badge label text                 |

### BadgeVariant

`'new'` | `'beta'` | `'deprecated'` | `'experimental'` | `'stable'` | `'info'` | `'warning'`
